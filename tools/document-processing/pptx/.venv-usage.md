# Virtual Environment - PPTX Skill

Este skill usa um **virtual environment Python** para isolar dependências.

## ✅ Virtual Environment Ativado

O virtual environment já foi criado em `.venv/` e contém todas as dependências necessárias.

### Como usar os scripts Python

**SEMPRE ative o virtual environment antes de usar os scripts:**

```bash
# Navegar até o diretório do skill
cd /Users/luizfosc/aios-core/.aios/skills/document-processing/pptx

# Ativar virtual environment
source .venv/bin/activate

# Agora use os scripts normalmente
python scripts/inventory.py example.pptx output.json
python scripts/thumbnail.py example.pptx thumbnails/ --cols 4
python ooxml/scripts/unpack.py example.pptx extracted/

# Desativar quando terminar
deactivate
```

### Atalho: Wrapper Scripts

Para facilitar, você pode criar wrappers que ativam o venv automaticamente:

```bash
# Criar wrapper para scripts principais
cat > run-pptx.sh << 'EOF'
#!/bin/bash
# Wrapper para scripts PPTX com venv
SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
source "$SCRIPT_DIR/.venv/bin/activate"
python "$@"
deactivate
EOF

chmod +x run-pptx.sh

# Usar:
./run-pptx.sh scripts/thumbnail.py example.pptx out/ --cols 4
```

### Adicionar Mais Dependências

```bash
source .venv/bin/activate
pip install nome-do-pacote
pip freeze > requirements.txt  # Atualizar requirements
deactivate
```

### Recriando o Virtual Environment

Se precisar recriar:

```bash
rm -rf .venv
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
deactivate
```
