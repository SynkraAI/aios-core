01
O conceito — Uma base, múltiplos humores
Como o Tony Robbins: "Liberdade dentro de um framework". A base nunca muda. O contexto colore.
O que NUNCA muda:
Preto #212121
Branco #FFFFFF
Amber #B8860B
Montserrat + Ubuntu
Chevron Pattern
O que FLEXIONA por contexto:
→ Fotos trazem suas próprias cores (palco quente, bastidor frio, evento colorido)
→ Lançamentos podem ter uma cor de campanha temporária que vive junto com o amber
→ O fundo pode variar entre preto puro, preto quente, preto frio, ivory, creme
→ A intensidade do chevron pattern sobe ou desce conforme o contexto
