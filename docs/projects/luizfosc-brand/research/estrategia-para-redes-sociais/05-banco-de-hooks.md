# Banco de Hooks

> "Os primeiros 3 segundos são o truque. O resto é o método. Se o truque não funciona, ninguém fica pra ver o método."

---

## Como Usar Este Banco

1. **Escolha o pilar** do conteúdo que vai produzir
2. **Filtre pela plataforma** onde vai publicar
3. **Selecione o tipo de hook** que combina com o formato
4. **Adapte o hook** ao conteúdo específico (nunca use literalmente sem contextualizar)

**Legenda de tipos:**
- 🔀 **Paradoxo** — inverte expectativa (estilo Petry)
- 🔢 **Número + Bastidor** — dado concreto + revelação (estilo Beirão)
- 🎬 **Cena Cinematográfica** — situa num momento vívido (estilo Rony)
- 💬 **Confissão Vulnerável** — revela fragilidade real
- ⚡ **Provocação Direta** — desafia crença sem rodeios

## Como Expandir Este Banco

### Método para criar novos hooks:
1. **Pegar um resultado/fato específico** da vida do Fosc
2. **Aplicar um dos 5 tipos** de hook (paradoxo, número, cena, confissão, provocação)
3. **Testar com a pergunta:** "Se eu lesse isso scrollando, eu pararia?"
4. **Validar com a regra:** "Esse hook poderia ser de qualquer palestrante?" Se sim, reescrever.

### Fontes de novos hooks:
- Cada palestra nova gera 3-5 hooks
- Cada podcast/entrevista gera 2-3 hooks
- Cada bastidor relevante gera 1-2 hooks
- Cada livro lido gera 1 hook

**Meta:** Adicionar 5+ hooks por mês ao banco. Revisar performance trimestral e promover os melhores.
