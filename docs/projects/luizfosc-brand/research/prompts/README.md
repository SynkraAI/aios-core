# Prompts — Produção de Conteúdo Luiz Fosc

Prompts prontos para gerar conteúdo alinhado com a estratégia, ICPs e tom de voz da marca.

## Como usar

Cada prompt é um arquivo `.md` que pode ser usado de 3 formas:
1. **Direto no Claude Code** — copiar e executar
2. **Via `/content-forge`** — o skill orquestra automaticamente
3. **Como referência** — adaptar para outras ferramentas

## Organização

| Arquivo | O que gera |
|---------|------------|
| `reels.md` | Roteiros de Reels (60-90s) por pilar |
| `carrosseis.md` | Estrutura de carrosséis educativos (5-8 slides) |
| `linkedin.md` | Posts narrativos e frameworks pro LinkedIn |
| `stories-textuais.md` | Stories textuais longos (formato Marsili) |
| `lives.md` | Roteiro-guia para lives semanais |
| `shorts-youtube.md` | Roteiros de Shorts/trechos de palestra |
| `opiniao-impopular.md` | Série "Opinião Impopular" |
| `desmontando-ilusao.md` | Série "Desmontando a Ilusão" |

## Regra de ouro

Antes de usar qualquer prompt, passar o resultado pelos 3 testes:
1. **Teste do Seguidor Fiel** — "Ele republicaria isso?"
2. **Teste Renner** — "Um guru genérico falaria isso?" Se sim, reescrever
3. **Teste de profundidade** — "Alguém inteligente respeitaria isso?"
