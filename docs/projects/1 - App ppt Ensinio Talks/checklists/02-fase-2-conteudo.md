## 🚨 Troubleshooting

### AI Studio começou a resumir / simplificar / esquecer
**Causa:** Contexto saturado (ironicamente o tema do slide 3.3).
**Solução:**
1. Exportar código atual do AI Studio
2. Abrir nova sessão
3. Colar código atual com prompt: *"Este é o estado atual do app. Continue a partir daqui. Vou enviar o conteúdo do próximo Ato."*
4. Continuar do Ato onde parou

### Um Ato ficou diferente do esperado
**Causa:** Prompt genérico ou ambíguo.
**Solução:**
1. Reverter o último prompt no AI Studio (ou undo)
2. Reler o conteúdo do Ato no roteiro
3. Adicionar instruções MAIS específicas ao prompt antes de recolar
4. Refazer

### Componentes quebraram
**Causa:** AI Studio "melhorou" algo que não devia.
**Solução:** Pedir explicitamente: *"NÃO altere os componentes existentes em components/. Só adicione conteúdo aos slides do Ato X."*
