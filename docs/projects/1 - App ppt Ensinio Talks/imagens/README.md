## 📝 Convenções de Naming

- Formato: `NN-nome-descritivo.png` onde NN é o número (01-12)
- Minúsculas, sem acentos, separado por hífens
- Sempre PNG (não JPG — qualidade + transparência possível)
- Resolução recomendada: 1920×1080 (16:9)
