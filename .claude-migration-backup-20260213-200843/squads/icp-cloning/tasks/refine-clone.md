# refine-clone

## Objetivo
Refinamento direcionado de áreas específicas

## Processo
Baseado em falhas de validação:
1. Identificar área problemática
2. Revisar dados fonte
3. Ajustar componente afetado
4. Re-validar

## Iterativo
Repetir até score >= 90/100

## Output
`outputs/refinement/REFINEMENT_LOG.md`

## Tempo: Variável (30 min - 2 horas)
