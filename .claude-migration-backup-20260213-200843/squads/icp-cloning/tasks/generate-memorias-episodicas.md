# generate-memorias-episodicas

## Objetivo
Gerar 10+ Memórias Episódicas

## Conceito
Episódios formadores com carga emocional

## Processo
Criar 10-20 memórias:
- Formação, Fracasso, Sucesso
- Relacionais, Virada, Dolorosas, Insight

## Output
`outputs/humanization/MEMORIAS_EPISODICAS.md`

## Tempo: 30-40 min
