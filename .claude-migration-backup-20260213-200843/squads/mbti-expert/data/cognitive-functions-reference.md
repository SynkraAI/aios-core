Source: Apostila GD (Modulos 1-3), Tese UFSCar (Belonsi, 2009)

**IMPORTANT:** The 16Personalities model does NOT use cognitive functions.
This reference covers the CLASSICAL Jungian/Myers-Briggs model only.

## The 8 Cognitive Functions

### Perceiving Functions (Irrational â€” how we take in information)

**Se â€” Extraverted Sensing (Sensacao Extrovertida)**
- Focus: Concrete, tangible reality in the present moment
- Characteristics: Observant, action-oriented, sensory-aware, present-focused
- Strengths: Quick reflexes, practical awareness, aesthetic appreciation
- Types with dominant/auxiliary: ESTP (dom), ESFP (dom), ISTP (aux), ISFP (aux)

**Si â€” Introverted Sensing (Sensacao Introvertida)**
- Focus: Internal impressions, past experiences, personal associations
- Characteristics: Detail-oriented, traditional, reliable, memory-focused
- Strengths: Recall of details, pattern recognition from experience, consistency
- Types with dominant/auxiliary: ISTJ (dom), ISFJ (dom), ESTJ (aux), ESFJ (aux)

**Ne â€” Extraverted Intuition (Intuicao Extrovertida)**
- Focus: External possibilities, connections between ideas, what could be
- Characteristics: Innovative, brainstorming, sees patterns across domains
- Strengths: Idea generation, seeing possibilities, connecting disparate concepts
- Types with dominant/auxiliary: ENTP (dom), ENFP (dom), INTP (aux), INFP (aux)

**Ni â€” Introverted Intuition (Intuicao Introvertida)**
- Focus: Internal vision, convergent insights, future implications
- Characteristics: Visionary, strategic, sees underlying meanings
- Strengths: Long-range planning, pattern synthesis, "aha" moments
- Types with dominant/auxiliary: INTJ (dom), INFJ (dom), ENTJ (aux), ENFJ (aux)

### Judging Functions (Rational â€” how we make decisions)

**Te â€” Extraverted Thinking (Pensamento Extrovertido)**
- Focus: External logic, efficiency, objective organization
- Characteristics: Systematic, decisive, results-oriented, procedural
- Strengths: Project management, organization, logical analysis of external data
- Types with dominant/auxiliary: ENTJ (dom), ESTJ (dom), INTJ (aux), ISTJ (aux)

**Ti â€” Introverted Thinking (Pensamento Introvertido)**
- Focus: Internal logic, precision, categorical frameworks
- Characteristics: Analytical, precise, theory-building, independent reasoning
- Strengths: Deep analysis, logical consistency, theoretical frameworks
- Types with dominant/auxiliary: INTP (dom), ISTP (dom), ENTP (aux), ESTP (aux)

**Fe â€” Extraverted Feeling (Sentimento Extrovertido)**
- Focus: External harmony, group values, social dynamics
- Characteristics: Empathetic, socially aware, harmonizing, expressive
- Strengths: Group facilitation, conflict resolution, emotional awareness of others
- Types with dominant/auxiliary: ENFJ (dom), ESFJ (dom), INFJ (aux), ISFJ (aux)

**Fi â€” Introverted Feeling (Sentimento Introvertido)**
- Focus: Internal values, authenticity, personal ethics
- Characteristics: Value-driven, authentic, idealistic, deeply personal
- Strengths: Moral clarity, artistic expression, empathy through identification
- Types with dominant/auxiliary: INFP (dom), ISFP (dom), ENFP (aux), ESFP (aux)

## Function Dynamics (from Apostila GD)

### Dominant Function
- Most developed and preferred
- Center of conscious personality
- For Extraverts: oriented toward the external world
- For Introverts: oriented toward the internal world
- Develops first (childhood)

### Auxiliary Function
- Balances the dominant
- Always opposite orientation (E<->I)
- Provides perception if dominant is judging, or vice versa
- Develops in adolescence

### Tertiary Function
- Same orientation as dominant
- Less developed; can be a source of growth
- Develops in midlife (30s-40s)

### Inferior Function
- Least developed; opposite of dominant
- Source of stress and "grip" experiences
- Under extreme stress, this function takes over in an immature way
- Portal to the unconscious; development opportunity in later life

### Shadow Functions (5th-8th)
The shadow is the unconscious complement of the conscious function stack.
Each type has 4 shadow functions that mirror the primary stack with opposite orientation:

| Position | Name | Relationship |
|----------|------|-------------|
| 5th | Opposing | Same function as dominant, opposite orientation |
| 6th | Critical Parent | Same function as auxiliary, opposite orientation |
| 7th | Trickster | Same function as tertiary, opposite orientation |
| 8th | Demon | Same function as inferior, opposite orientation |

## Stress and Grip (Inferior Function Activation)

When under extreme stress, each type "falls into the grip" of their inferior function:

| Type | Inferior | Grip Behavior |
|------|----------|---------------|
| INTJ/INFJ | Se | Overindulgence in sensory pleasures, impulsive behavior |
| INTP/INFP | Fe/Te | Emotional outbursts (INTP) or harsh criticism (INFP) |
| ENTJ/ENFJ | Fi/Ti | Hypersensitivity (ENTJ) or obsessive logical analysis (ENFJ) |
| ENTP/ENFP | Si | Obsession with bodily symptoms, dwelling on past failures |
| ISTJ/ISFJ | Ne | Catastrophizing, seeing only negative possibilities |
| ESTJ/ESFJ | Fi/Ti | Emotional vulnerability (ESTJ) or cold logical analysis (ESFJ) |
| ISTP/ISFP | Fe/Te | Emotional outbursts (ISTP) or controlling behavior (ISFP) |
| ESTP/ESFP | Ni | Paranoid visions of doom, conspiracy thinking |

## Jung's Key Concepts (from Apostila GD)

### Psyche Structure
- **Ego:** Center of consciousness; thoughts and conscious memory
- **Personal Unconscious:** Repressed content, complexes
- **Collective Unconscious:** Archetypes shared by humanity
- **Self:** Totality of personality; integration goal

### Key Archetypes
- **Persona:** Social mask; how we present to the world
- **Shadow:** Repressed opposite of persona
- **Anima/Animus:** Contrasexual archetype (feminine in men / masculine in women)
- **Self:** Integration of all aspects; individuation goal

### Individuation Process
The process of becoming a complete, integrated Self by:
1. Confronting the Shadow
2. Integrating Anima/Animus
3. Developing inferior functions
4. Achieving union of opposites (transcendent function)

### Compensation Principle
- Unconscious compensates for conscious one-sidedness
- If consciousness is overly rational -> unconscious produces emotional content
- Dreams as compensatory messages from the unconscious
