# Checklist: Sources Validation

> BLOCKING GATE - All criteria must pass before extraction can proceed.

---

## Minimum Criteria

- [ ] **10+ total sources** across all tiers
- [ ] **5+ Tier 1 sources** (primary, by the expert)
- [ ] **3+ different source types** (book, interview, article, video, course, podcast)
- [ ] **5+ hours OR 200+ pages** of total content volume
- [ ] **Main framework triangulated** (confirmed by 3+ independent sources)

## Quality Signals

- [ ] At least 1 source is a **book written by the expert**
- [ ] At least 1 source is a **long-form interview** (45+ min)
- [ ] At least 2 sources are **unscripted** (podcasts, Q&A, live)
- [ ] Sources span **multiple years** (not all from same period)
- [ ] **Gold/Bronze ratio** favors gold content

## Decision

| Checks | Status | Action |
|--------|--------|--------|
| 5/5 minimum + 3+ quality | **GO** | Proceed to extraction |
| 5/5 minimum + <3 quality | **GO** | Proceed, note quality gaps |
| 4/5 minimum | **CONDITIONAL** | Proceed with acquisition plan |
| <4/5 minimum | **NO-GO** | Stop, acquire more sources |
| <5 total sources | **FAIL** | Expert too obscure |
