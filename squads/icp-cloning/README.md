# 🧬 ICP Cognitive Cloning Squad

Sistema de clonagem cognitiva profunda de ICPs (Ideal Customer Profiles) com 100% de eficácia.

## 📊 Overview

- **Framework:** SYNAPSE v6.0 (TSM • APR • MCC • PSH)
- **Tempo:** 12-16 horas (completo) | 6-8 horas (simplificado)
- **Output:** System Prompt (1000+ linhas) + 12 Knowledge Base docs
- **Fidelidade:** 100% (validado com clone "Marina")

## 🎯 O Que Este Squad Faz

Transforma dados básicos de persona em clone conversacional humanizado que:
- Pensa como a persona (frameworks cognitivos)
- Fala como a persona (assinatura linguística - 1500+ exemplos)
- Decide como a persona (heurísticas + custo mental)
- Sente como a persona (triggers emocionais + memórias)
- Se adapta como a persona (plasticidade comportamental)

## 🏗️ Arquitetura

### 3 Camadas de Processamento

| Camada | Agents | Função | Tempo |
|--------|--------|--------|-------|
| **1. Extração** | extractor-deep | 15 prompts (P0-P14) | 4-5h |
| **2. Processamento** | synapse-processor, humanizer | SYNAPSE + Humanização | 4-6h |
| **3. Geração** | clone-generator, clone-validator | Outputs + QA | 3-5h |

**Orquestrador:** synapse-architect (entry point)

## 📋 Processo em 6 Fases

### Fase 1: Entrada (30 min)
Preencher briefing ICP com dados básicos.

### Fase 2: Extração Profunda (4-5h)
Executar 15 prompts de extração (P0-P14):
- **P0-P0B:** Demografia + Financeiro
- **P1-P3:** Psicologia (Big Five, Linguagem, Valores)
- **P4-P6:** Social (Comunidade, Triggers, Digital)
- **P7-P9:** Decisão (Compra, Narrativas, Expectativas)
- **P10-P14:** Decision Intelligence

### Fase 3: Processamento SYNAPSE (2-3h)
Aplicar Framework SYNAPSE v6.0:
- **TSM:** 8 Trigger-State-Manifestation triplets
- **APR:** 5 Adaptive Plasticity Rules
- **MCC:** 7 Mental Computational Cost assessments
- **PSH:** Psychographic Shell (6 camadas integradas)

### Fase 4: Humanização Avançada (2-3h)
Adicionar 6 camadas de autenticidade:
- Blind Spots (8 pontos cegos)
- Paradoxos Produtivos (6 contradições)
- Fingerprints Únicos (7 assinaturas)
- Memórias Episódicas (10+ memórias)
- Sistema Imunológico (3 níveis)
- Meta-Axiomas (9 princípios)

### Fase 5: Geração Final (2-3h)
Gerar outputs finais:
- **System Prompt:** 1000+ linhas executáveis
- **Knowledge Base:** 12 documentos especializados

### Fase 6: Validação (1-2h)
Garantir qualidade 100%:
- 30 perguntas conversacionais
- Checklist 100 pontos
- Fidelity score (5 dimensões)
- Edge case testing

## 📊 Framework SYNAPSE v6.0

### TSM (Trigger-State-Manifestation)
Mapeia cadeias comportamentais completas:
```
Estímulo Externo → Estado Interno → Manifestação Comportamental
```

### APR (Adaptive Plasticity Rules)
Modela como a persona se adapta a novos contextos.

### MCC (Mental Computational Cost)
Quantifica custo cognitivo de decisões e mudanças.

### PSH (Psychographic Shell)
Integração final de todos componentes em estrutura executável.

## 🔧 Manutenção e Evolução

### Quando Atualizar Clone

- **Minor updates:** A cada 3-6 meses (novos dados)
- **Major updates:** A cada 1-2 anos (mudanças significativas)
- **Emergency:** Drift crítico detectado

### Workflow de Atualização

```
*update-clone
```

Processo:
1. Diagnóstico de drift
2. Re-extração direcionada (apenas camadas afetadas)
3. Re-processamento parcial
4. Merge com versão anterior
5. Re-validação
