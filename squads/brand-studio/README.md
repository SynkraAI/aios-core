# Brand Studio Squad

> "A brand is not what you say it is. It's what they say it is." — Marty Neumeier

Squad de branding completo que opera como uma agência de elite. Cada especialista é baseado nos maiores nomes do mundo em suas áreas, com seus frameworks exclusivos e formas de pensar únicas.

## O Squad

| Agente | Persona | Baseado em | Papel |
|--------|---------|------------|-------|
| `@marty-neumeier` | 🎯 Chief | Marty Neumeier | Brand Chief — Direção criativa, integração final |
| `@david-aaker` | 📋 Account Director | David Aaker | Briefing, brand equity, distribuição de trabalho |
| `@simon-sinek` | 🔵 Purpose Strategist | Simon Sinek | Propósito, WHY, posicionamento, manifesto |
| `@alexandra-watkins` | ✍️ Naming Expert | Alexandra Watkins | Naming, SMILE/SCRATCH, shortlist de nomes |
| `@paula-scher` | 🎨 Visual Identity | Paula Scher | Identidade visual, logo, tipografia, cor |
| `@ann-handley` | 🗣️ Brand Voice | Ann Handley | Voz de marca, messaging, taglines |
| `@malcolm-gladwell` | 🔍 Consumer Insights | Malcolm Gladwell | Pesquisa, insights culturais, audiência |

```
Cliente → @marty-neumeier (intake)
              ↓
         @david-aaker (briefing completo)
              ↓
    ┌─────────┼─────────┐
    ↓         ↓         ↓

    ↓         ↓
```

## Como Usar

### Projeto Completo de Marca
```
@marty-neumeier *new-brand
```
Marty coleta o escopo inicial e inicia o fluxo completo de agência.

### Entrar Direto num Especialista
```
@alexandra-watkins *name-brand
@paula-scher *brand-identity
@simon-sinek *find-why
@ann-handley *brand-voice
@malcolm-gladwell *audience-research
@david-aaker *brand-briefing
```

### Revisar uma Marca Existente
```
@marty-neumeier *brand-audit
```

## Frameworks Utilizados

| Especialista | Framework Exclusivo |
|-------------|---------------------|
| Marty Neumeier | Five Disciplines (Differentiate, Collaborate, Innovate, Validate, Cultivate) + ZAG Test |
| David Aaker | Brand Equity Model (Awareness, Associations, Quality, Loyalty, Assets) |
| Simon Sinek | Golden Circle (WHY → HOW → WHAT) |
| Alexandra Watkins | SMILE Test + SCRATCH Test |
| Paula Scher | Visual language as brand ambassador — typography-first |
| Ann Handley | VOICE Framework (Voice, Optimized, Insightful, Cultural, Engaging) |
| Malcolm Gladwell | Tipping Point (Mavens, Connectors, Salesmen) + Thin-Slicing (Blink) |

## Entregáveis por Projeto Completo

1. **Brand Briefing Document** — Aaker
2. **Consumer Insights Report + Audience Archetypes** — Gladwell
3. **Brand Purpose Statement + Manifesto** — Sinek
4. **Brand Name Candidates (3-5) + Rationale** — Watkins
5. **Visual Identity Brief + Brand Guidelines Direction** — Scher
6. **Brand Voice Guide + Core Messaging** — Handley
7. **Final Brand System Report** — Neumeier (integração)

## Instalação

Este squad é parte do repositório `aios-core`. Para usar em seu projeto:

```bash
npx aios-core install
# Selecionar "brand-studio" durante a instalação
```
