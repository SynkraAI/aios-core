# Brand Briefing — David Aaker

Coleta o briefing completo usando o Brand Equity Framework de Aaker.
Entrega o Brand Brief e distribui contexto customizado para cada especialista.

## Framework de Coleta

### Bloco 1: Empresa & Contexto

```
1. Qual é o negócio central? Como a empresa faz dinheiro?

2. Há quanto tempo no mercado? Qual é a história da fundação?

3. Quais são os marcos mais importantes até hoje?

4. Qual é o tamanho atual (revenue, clientes, equipe)?

5. Quais são os planos de crescimento (próximos 3 anos)?
```

### Bloco 2: Brand Equity Atual (apenas para marcas existentes)

**Brand Awareness:**
```
- Qual é o nível de reconhecimento espontâneo da marca?
- Em quais mercados geográficos/segmentos são conhecidos?
- Já fizeram pesquisa de brand awareness? Resultados?
```

**Brand Associations:**
```
- Quando clientes pensam na marca, o que vem à mente?
- Quais associações são desejadas vs. existentes (e diferentes)?
- O que dizem os reviews, NPS, pesquisas de satisfação?
```

**Perceived Quality:**
```
- Como sua qualidade é percebida vs. competidores?
- Há gap entre qualidade real e percebida?
```

**Brand Loyalty:**
```
- Qual é a taxa de retenção de clientes?
- O que faria um cliente fiel migrar para o concorrente?
- Há advocates/evangelizadores naturais?
```

**Proprietary Brand Assets:**
```
- Quais ativos de marca já existem? (logo, cores, slogans registrados)
- Há IP, trade dress, ou elementos visuais distintos?
```

### Bloco 3: Audiência & Mercado

```
1. Descreva o cliente ideal — quem é ele?
   (Além de demografia: comportamento, crenças, aspirações)

2. Quem é o cliente secundário (influenciador, pagador, usuário diferente)?

3. Quais são as maiores dores/frustrações da audiência com a categoria?

4. O que a audiência ama/odeia nos competidores?

5. Como sua audiência descobre marcas novas? (canais, influências)
```

### Bloco 4: Competidores

```
1. Liste os 3-5 competidores diretos mais relevantes.

2. Para cada um:
   - Qual é o posicionamento deles?
   - O que fazem bem?
   - Onde são fracos?
   - O que você tem que eles não têm?

3. Há competidores indiretos (categoria adjacente) que roubam atenção?

4. Há alguma marca fora da categoria que admira como referência?
```

### Bloco 5: Ambição & Métricas

```
1. Como seria o sucesso desta marca daqui a 2 anos?
   (Seja específico: market share, NPS, revenue, reconhecimento)

2. Quais métricas de marca importam mais para vocês?

3. Há KPIs de negócio que este projeto de marca deve suportar?
```

### Bloco 6: Restrições

```
1. Qual é o budget total do projeto?

2. Qual é o deadline? Há um evento/lançamento que ancora?

3. Há elementos da marca atual que DEVEM ser mantidos? Por quê?

4. Há restrições legais, regulatórias, ou de trademark já identificadas?

5. Há stakeholders internos que precisam aprovar? Quem tem veto?

6. Há sensibilidades culturais ou de mercado que precisamos respeitar?
```
