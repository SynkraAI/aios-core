# Data Directory

This directory is reserved for runtime data used by the mind-content-updater squad.

## Possible contents

- Cached search results (to avoid re-searching)
- URL deduplication databases
- Collection progress state files

## Note

Source data is NOT stored here. All collected sources go to:
`squads/mmos-squad/minds/{mind_slug}/sources/`

Reports and logs go to:
`squads/mmos-squad/minds/{mind_slug}/docs/logs/`
