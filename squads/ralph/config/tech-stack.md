# Ralph Squad - Tech Stack

## Runtime
- Node.js 18+ (scripts)
- Claude Code CLI (orchestration)

## Core Dependencies
- AIOS Core (agent system, task execution)
- Task tool (subagent delegation)
- Git (auto-commit)

## File Formats
- YAML: State files (ralph-state.yaml, ralph-config.yaml)
- Markdown: Progress tracking (progress.md, decision-log.md)
- Markdown: Stories and PRDs (source files)

## Integration Points
- All AIOS agents (@dev, @qa, @architect, @pm, @po, @sm, @analyst, @devops, @data-engineer, @ux-design-expert)
- Git CLI (auto-commit)
- Story/PRD system (checkbox tracking)
