## Overview

O Creator OS e um squad de 5 agentes especializados + 1 orquestrador que formam um sistema completo para:

1. **Identificar tendencias** e destilar conhecimento (Tiago Forte)
2. **Criar conteudo** que prende atencao (Nicolas Cole)
3. **Distribuir em escala** com repurposing automatizado (Sahil Bloom)
4. **Monetizar** via produto digital (Dan Koe)
5. **Sistematizar** tudo em um Content OS (Justin Welsh)

---

## Quick Start

### Comandos Principais

| Comando | O que faz |
|---------|-----------|
| `*start` | Setup inicial - diagnostico + Content OS |
| `*content {topic}` | Pipeline completo: pesquisa → escrita → distribuicao |
| `*write {type}` | Criar peca de conteudo especifica |
| `*research {topic}` | Pesquisar e destilar topico |
| `*monetize` | Desenhar modelo de negocio/oferta |
| `*distribute` | Plano de distribuicao |
| `*launch` | Pipeline de lancamento de produto |
| `*pipeline` | Pipeline completo (todos os agentes) |
| `*status` | Estado atual do sistema |
| `*help` | Lista de comandos |

### Primeiro Uso

```
1. Ative o Creator OS Chief
2. Execute *start para o diagnostico inicial
3. Use *content "seu topico" para criar seu primeiro conteudo
```
