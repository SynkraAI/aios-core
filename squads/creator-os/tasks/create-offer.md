# Task: Create Digital Offer

**Task ID:** create-offer
**Type:** Hybrid (interactive for validation, autonomous for design)
**Agent:** Dan Koe
**Squad:** Creator OS

---

## Purpose

Desenhar uma oferta digital irresistivel com product stack completo, pricing baseado em resultado, e modelo de receita recorrente.

## Inputs

| Input | Source | Required |
|-------|--------|----------|
| Creator profile | Genius Zone Blueprint | Yes |
| Nicho | Justin Welsh diagnostic | Yes |
| Audiencia data | Sahil Bloom metrics | No |
| Revenue goal | Blueprint (50k+/mes) | Yes |

## Process

### Step 1: Diagnostico
Definir:
- Expertise principal do creator
- Resultado que entrega (outcome claro)
- Audiencia-alvo (quem paga por esse resultado)
- Problema urgente que resolve

### Step 2: Offer Design (Value Equation)
Aplicar a Hormozi Value Equation:
```
Value = (Dream Outcome x Perceived Likelihood) / (Time Delay x Effort & Sacrifice)
```

Desenhar oferta que:
- Maximiza Dream Outcome (resultado aspiracional)
- Aumenta Perceived Likelihood (prova social, garantia)
- Minimiza Time Delay (resultado rapido)
- Minimiza Effort (facil de implementar)

### Step 3: Product Stack (Escada de Valor)
```
Gratuito: Posts + Newsletter (captura atencao e email)
R$47-97: E-book/Template (filtro de compradores)
R$297-997: Curso online (transformacao guiada)
R$2k-5k: Mentoria em grupo (acesso + comunidade)
R$5k-15k: Consultoria 1:1 (premium, poucos clientes)
```

### Step 4: Revenue Model
Calcular path para meta de receita:
```
Meta: 50k/mes
Opcao A: 100 alunos x R$497 (curso) = R$49.700/mes
Opcao B: 20 mentorados x R$2.500/tri = R$50k/tri = ~R$16.7k/mes + curso
Opcao C: Mix (curso + mentoria + templates)
```

### Step 5: Channel Strategy
- Organico vs Pago
- Funil evergreen vs lancamento
- Automacao de vendas

## Quality Gate

- Oferta tem resultado claro (outcome, nao features)
- Pricing baseado em valor, nao em custo
- Path to 50k/mes e matematicamente viavel
- Product stack tem pelo menos 3 niveis
- Automacao identificada para escalar sem equipe
