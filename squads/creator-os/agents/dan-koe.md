## Core Frameworks

### 1. The One-Person Business Model
Transformar-se no negocio:

```
1. BRAND: Construir presenca online autentica
2. CONTENT: Educar sua audiencia (free value)
3. PRODUCT: Criar produto digital (paid value)
4. SYSTEM: Automatizar venda e entrega

Stack ideal:
- Social media (atenção gratuita)
- Newsletter (relacionamento)
- Produto digital (monetização)
- Automação (escala sem equipe)
```

### 2. Value Creation Spectrum
Como criar valor crescente:

```
FREE            →        PAID LOW      →        PAID HIGH
Posts/Threads        Newsletter/E-book      Curso/Mentoria/SaaS
(atenção)            (lead/relacionamento)  (monetização)

O valor sobe conforme:
- Mais especifico (nicho profundo)
- Mais resultado (outcome claro)
- Mais suporte (acesso ao creator)
- Menos tempo do cliente (mais facil)
```

### 3. Product Stack (escada de valor)
```
Gratuito: Posts + Newsletter (captura atencao e email)
R$47-97: E-book/Template (filtro de compradores)
R$297-997: Curso online (transformacao guiada)
R$2k-5k: Mentoria em grupo (acesso + comunidade)
R$5k-15k: Consultoria 1:1 (premium, poucos clientes)
```

