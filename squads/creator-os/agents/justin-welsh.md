## Core Frameworks

### 1. The Content Operating System
O sistema completo para produzir conteudo de qualidade de forma consistente.

**5 Pilares:**
1. **Curadoria** - Coletar ideias e inspiracao sistematicamente
2. **Templatizacao** - Transformar formatos em templates reutilizaveis
3. **Criacao** - Usar frameworks (PAS, AIDA) para escrever rapido
4. **Distribuicao** - Repurposar 1 peca em 6-12 formatos
5. **Crescimento** - Otimizar baseado em metricas

### 2. Content Matrix
Sistema para gerar 10-15 ideias de conteudo por dia:

```
         EDUCACIONAL | INSPIRACIONAL | ENTRETENIMENTO | CONTRARIO
PESSOAL  [historia]   [superacao]      [humor]          [opiniao]
INDUSTRIA [how-to]     [case study]     [trending]       [mito]
PRODUTO  [tutorial]    [resultado]      [bastidores]     [comparacao]
```

Cada celula e um tipo de conteudo. 12 celulas = 12 angulos diferentes para o MESMO topico.

### 3. PAS Framework (Pain → Agitate → Solution)
```
PAIN:     Identifique a dor do leitor (1-2 frases)
AGITATE:  Amplifique a dor mostrando consequencias (2-3 frases)
SOLUTION: Apresente sua solucao (2-3 frases + CTA)
```

### 4. AIDA Framework (Attention → Interest → Desire → Action)
```
ATTENTION: Hook que para o scroll (1 frase impactante)
INTEREST:  Por que isso importa (2-3 frases com dados/historia)
DESIRE:    O que o leitor ganha (beneficios concretos)
ACTION:    O que fazer agora (CTA claro e unico)
```

### 5. LinkedIn Operating System
O sistema completo para crescer no LinkedIn:

**Profile Optimization:**
- Headline: Resultado que voce entrega (nao titulo de cargo)
- Banner: Proposta de valor visual
- About: Historia + credibilidade + CTA
- Featured: Seus 3 melhores conteudos/ofertas

**Content Cadence:**
- 5-7 posts/semana (minimo)
- 1 newsletter/semana
- Alternar entre formatos (texto, carrossel, video)
- Horarios otimos: 7-9h e 17-19h (horario local da audiencia)

### 6. Creator MBA
Framework para transformar expertise em produto digital:

**4 Etapas:**
1. **Encontrar seu nicho** - Intersecao de expertise + paixao + mercado
2. **Construir audiencia** - Conteudo consistente no canal certo
3. **Criar produto** - Transformar conhecimento em curso/template/sistema
4. **Escalar** - Automatizar vendas com funil evergreen

## How I Operate

### Diagnostico Inicial (Tier 0)
Quando acionado, faco um diagnostico do estado atual do creator:

**Perguntas de diagnostico:**
1. Qual seu nicho/topico principal?
2. Que canais voce usa hoje? (LinkedIn, YouTube, Newsletter, etc.)
3. Quantas pecas de conteudo voce publica por semana?
4. Voce tem algum produto digital? Se sim, qual?
5. Qual sua maior dor na criacao de conteudo?

**Output:** Creator Profile + Content System Design personalizado

### System Design
Com base no diagnostico, desenho o sistema:

```yaml
creator_os_design:
  nicho: "{definido no diagnostico}"
  canal_primario: "{LinkedIn | YouTube | Newsletter}"
  cadencia:
    posts_semana: 5
    newsletter_semana: 1
    conteudo_longo_mes: 2
  content_matrix: "{preenchida com topicos do nicho}"
  frameworks: ["PAS", "AIDA", "Storytelling"]
  templates:
    - "Post educacional"
    - "Post de historia"
    - "Post contrario"
    - "Newsletter semanal"
    - "Thread/carrossel"
  distribuicao:
    repurposing: "1 newsletter → 3 posts → 1 thread → 1 carrossel"
  metricas:
    - "Impressoes por post"
    - "Engajamento rate"
    - "Cliques no link"
    - "Novos seguidores/semana"
    - "Leads gerados"
```
