## Core Frameworks

### 1. Newsletter Growth System
O sistema para crescer uma newsletter de forma consistente:

**Cadencia semanal:**
- Segunda + Sexta: Ideias curtas e actionable (momentum)
- Quarta: Deep dive com insights profundos (valor)
- Sabado: Week-in-review (catch-up para quem perdeu)

**Growth Levers:**
1. Cross-promotion com outras newsletters
2. Social media → Newsletter funnel
3. Referral program (subscribers trazem subscribers)
4. Guest appearances (podcasts, lives, collabs)
5. SEO evergreen (artigos que trazem traffic organico)

### 2. Content Loops
Cada peca de conteudo alimenta a proxima:

```
Newsletter → Post LinkedIn (excerpt)
Post LinkedIn → Comments → Novo topico
Novo topico → Newsletter (deep dive)
Newsletter → Thread Twitter (highlights)
Thread → Engajamento → Mais assinantes
Mais assinantes → Newsletter
... LOOP
```

### 3. Repurposing Matrix
1 peca longa → multiplos formatos:

```
1 Newsletter (1000 palavras)
├── 3 Posts LinkedIn (insights isolados)
├── 1 Thread Twitter (versao sequencial)
├── 1 Carrossel Instagram (versao visual)
├── 5 Stories (micro-insights)
└── 1 Short-form video (1 min highlight)
```

### 4. Distribution Calendar
```yaml
segunda: "Short insight post (LinkedIn + Twitter)"
terca: "Carrossel ou visual (Instagram + LinkedIn)"
quarta: "Newsletter deep dive (Email) + excerpt post"
quinta: "Thread (Twitter) + storytelling post (LinkedIn)"
sexta: "Friday Five curadoria (Newsletter + post)"
sabado: "Week-in-review (Newsletter)"
domingo: "Batch create conteudo da proxima semana"
```
