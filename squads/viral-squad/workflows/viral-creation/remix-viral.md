# 🔄 Remix Viral Hit — Workflow de Remixagem

## Objetivo

Remixar vídeo viral de competidor com twist único, criando conteúdo diferenciado que surfa na onda sem copiar.

## Quando Usar

- Competidor viralizou com conteúdo relevante para o seu nicho
- Trend está em ascensão e há janela de oportunidade
- Você quer capitalizar um formato comprovado com sua identidade

## Agentes

- 🔍 **@research-ninja** (líder) — reverse engineering do viral
- 🔥 **@viral** — estratégia de diferenciação
- 📝 **@script-architect** — roteiro remixado
- 💥 **@visual-impact** — identidade visual própria
- 🏗️ **@remotion-architect** — implementação técnica
- 🎬 **@art-director** — direção criativa do remix

## Workflow com Debate

### FASE 1: REVERSE ENGINEERING (4h)

**1.1 Análise do Viral Original**

```
@research-ninja: *reverse-engineer [URL do vídeo viral]

Output esperado:
REVERSE ENGINEERING REPORT
━━━━━━━━━━━━━━━━━━━━━━━━

Viral Stats:
  Views: 2.3M | Likes: 180K | Shares: 45K | Saves: 92K
  Viral Coefficient: 3.2
  Peak Period: primeiras 48h

Hook Analysis:
  Type: Pattern Interrupt + Curiosity Gap
  Copy: "Você NÃO deveria estar fazendo isso na academia"
  Visual: Close-up de exercício sendo feito errado
  Retention @3s: ~85% (estimado)

Script Structure:
  Format: PAS (Problem-Agitate-Solution)
  Duration: 47s
  Open Loops: 3 (0:03, 0:15, 0:32)
  Payoff: 0:38

Viral Elements (STEPPS):
  ✅ Social Currency: Alto (conhecimento exclusivo)
  ✅ Emotion: Surpresa + Raiva (exercício errado)
  ✅ Practical Value: Alto (correção prática)
  ⬚ Triggers: Médio (contexto de academia)
  ⬚ Public: Baixo (não visualmente compartilhável)
  ⬚ Stories: Baixo (sem narrativa forte)

Share Triggers:
  Primary: "Preciso mostrar isso pro meu amigo que faz errado"
  Secondary: "Eu sabia! Validação social"
```

**1.2 Identificação de Gaps**

```
@research-ninja: *competitor-gap [viral analisado]

DEBATE:
@research-ninja: "Viral focou só em homens jovens (18-25). Gap: público 30-45"
@research-ninja: "Formato PAS usado, mas sem Before/After visual"
@viral: "Stories score baixo — oportunidade de adicionar narrativa"
@viral: "Public score baixo — podemos tornar mais visualmente compartilhável"

CONSENSO: ✅ Gaps identificados:
  1. Audiência expandida (30-45)
  2. Narrativa pessoal (Stories element)
  3. Before/After visual (Public element)
```

---

### FASE 2: DIFERENCIAÇÃO (1 dia)

**2.1 Estratégia de Twist**

```
@viral: *viral-strategy [remix com diferenciação]

REGRA 70/30:
  70% similar: Manter elementos virais comprovados
    - Hook type (Pattern Interrupt)
    - Script structure (PAS)
    - Practical value (correção)
  30% único: Adicionar twist diferenciador
    - Narrativa pessoal ("Eu fazia errado por 5 anos")
    - Before/After visual real
    - Audiência expandida (linguagem mais madura)

DEBATE:
@viral: "Twist: adicionar transformação pessoal. STEPPS Stories sobe de Baixo para Alto"
@script-architect: "Concordo. Abre com vulnerabilidade: 'Eu fazia isso errado por anos'"
@research-ninja: "Competidor não tem credibilidade pessoal. Esse é nosso diferencial"

CONSENSO: ✅ Twist = narrativa pessoal + Before/After + audiência madura
```

**2.2 Roteiro Remixado**

```
@script-architect: *viral-script [remix]

REMIX SCRIPT (52s)

[0-3s] HOOK — Pattern Interrupt + Vulnerabilidade
  Copy: "Eu fiz esse exercício ERRADO por 5 anos. E ninguém me avisou."
  Visual: Close-up facial, expressão de frustração genuína
  Audio: Scratch sound → silêncio dramático

[3-12s] BEFORE — Setup da dor
  Copy: "Treino pesado, 6x por semana. Zero resultado."
  Visual: Montage de treinos antigos (estilo amador)
  Open Loop: "Até que um fisioterapeuta me mostrou..."

[12-30s] AGITATE — O erro revelado
  Copy: "Olha a diferença." [demonstração lado a lado]
  Visual: Split screen — errado vs correto, câmera lenta
  Open Loop: "E não é só esse exercício..."

[30-45s] SOLUTION — A correção completa
  Copy: "3 exercícios que eu corrigi. Resultado em 30 dias."
  Visual: Before/After real (fotos com timestamp)
  Evidence: Fotos reais com datas visíveis

[45-52s] CTA — Share trigger
  Copy: "Marca aquele amigo que treina errado 😂"
  Visual: Bold CTA + gold accent
  Share Trigger: Tag social (obrigação social de avisar)

DEBATE:
@hook-master: "Hook combina Pattern Interrupt + vulnerabilidade. Score: 93/100"
@retention-architect: "Open loops nos segundos 8 e 25 garantem retenção"
@script-architect: "Before/After real é prova social impossível de ignorar"

CONSENSO: ✅ Script aprovado
```

---

### FASE 3: PRODUÇÃO (1 dia)

**3.1 Direção Criativa**

```
@art-director: *art-direct [remix script]

CREATIVE BRIEF — Remix Fitness
  Visual Direction: Documentary-style + split screens
  Differentiation: Fotos reais, não stock
  DS: Design System configurado enforced
  Gold Usage: CTA button + timestamp highlights (< 8%)

Pipeline Paralelo:
  Track A: @visual-impact → First frame + visual hooks
  Track B: @motion-master → Split screen transitions
  Track C: @remotion-architect → Component implementation
  Track D: @sound-designer → Audio design
```

**3.2 Execução**

```
@remotion-architect: Implementar composição
@visual-impact: Validar que execução visual > original
@render-master: Render final

QUALITY GATE:
  - Visualmente superior ao original? ✅/❌
  - Twist diferenciador claro? ✅/❌
  - DS compliance? ✅/❌
  - 60fps? ✅/❌
```

---

### FASE 4: LANÇAMENTO ESTRATÉGICO (2h)

```
@algorithm-hacker: *launch-strategy [remix]

Timing:
  - Publicar enquanto trend do original ainda está ativa
  - Ideal: 24-72h após o pico do original
  - Evitar: > 7 dias (trend morta)

Caption Strategy:
  - NÃO mencionar o original (evitar comparação direta)
  - Focar na narrativa pessoal
  - CTA de tag (share trigger)

Post-Publish (primeiras 2h):
  - @engagement-engineer: Reply ALL comments < 5min
  - Share to Story com poll: "Você faz esse exercício certo?"
  - DM 20 seguidores engajados
```

## Métricas de Sucesso

| Métrica | Target | vs Original |
|---------|--------|-------------|
| Retention @3s | > 80% | ≥ original |
| Save Rate | > 10% | > original (practical value + pessoal) |
| Share Rate | > 5% | > original (tag CTA) |
| Viral Coefficient | > 2.0 | Comparável |

## Checklist Pré-Lançamento

- [ ] Reverse engineering completo do original
- [ ] Twist 30% claramente diferenciador
- [ ] Roteiro aprovado pelo squad
- [ ] Execução visual superior ao original
- [ ] DS compliance validado
- [ ] Timing de lançamento dentro da janela da trend
- [ ] Caption e hashtags otimizados
- [ ] Post-publish plan pronto

**Tempo Total: 2-3 dias**
**Output:** Remix com identidade própria que surfa na onda do original
