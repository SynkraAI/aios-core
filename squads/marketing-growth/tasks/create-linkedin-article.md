# Create LinkedIn Article

**Task:** `create-linkedin-article`
**Agent:** @content-producer + @lead-copywriter
**Type:** Creation

---

## Description

Criar artigo LinkedIn long-form (800-2000 palavras) SEO-optimized.

---

## Input

```yaml
tema: 'Tema do artigo'
palavras_chave: ['keyword1', 'keyword2']
extensao: '800 | 1200 | 1500 | 2000'
```

---

## Process

### 1. Estrutura (@lead-copywriter)

- Define estrutura argumentativa
- Headings H2/H3

### 2. Escrita (@content-producer)

- Artigo SEO-friendly
- Introducao com hook
- Corpo com subtitulos
- Conclusao com CTA

### 3. Quality Gate

- Checklists: seo-basics + content-quality

---

## Output

```
ARTIGO LINKEDIN:
- Titulo (SEO-optimized)
- Imagem de capa (spec 1200x644px)
- Corpo completo com headings
- 3-5 hashtags
```
