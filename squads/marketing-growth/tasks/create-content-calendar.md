# Create Content Calendar

**Task:** `create-content-calendar`
**Agent:** @content-strategist
**Type:** Creation

---

## Description

Criar calendario editorial mensal (30 dias x 2 plataformas).

---

## Input

```yaml
mes: 'YYYY-MM'
temas_principais: ['tema1', 'tema2', 'tema3']
eventos: 'Eventos, feriados, lancamentos do mes'
metas_mensais: 'Metas de followers, engagement, conversao'
```

---

## Process

1. Define pilares de conteudo (3-5)
2. Distribui temas por semana
3. Define formato por dia (carousel, reel, post, article)
4. Alterna plataformas (IG/LI)
5. Marca datas importantes

---

## Output

Tabela markdown 30 dias com: dia, plataforma, formato, tema, hook sugerido, status.
