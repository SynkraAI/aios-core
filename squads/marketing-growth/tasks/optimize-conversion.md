# Optimize Conversion

**Task:** `optimize-conversion`
**Agent:** @conversion-optimizer
**Type:** Optimization

---

## Description

Otimizar taxa de conversao de pagina ou funil especifico.

---

## Input

```yaml
tipo: 'lp | form | cta | funnel'
url_ou_content: 'URL ou HTML da pagina'
metrica_atual: 'Taxa de conversao atual (se conhecida)'
```

---

## Process

1. Audit CRO (7 pontos)
2. Prioriza por impacto x esforco (PIE)
3. Implementa mudancas
4. Cria plano A/B test
5. Define metricas de sucesso

---

## Output

```
Versao otimizada + justificativas
Plano A/B test (hipotese + variantes + metricas)
Estimativa de melhoria
```
