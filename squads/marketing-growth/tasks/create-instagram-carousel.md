# Create Instagram Carousel

**Task:** `create-instagram-carousel`
**Agent:** @content-producer + @visual-designer
**Type:** Creation

---

## Description

Criar carrossel Instagram com 7-10 slides educativos, hook forte e CTA.

---

## Input

```yaml
tema: 'Tema do carrossel'
angulo: 'educativo | storytelling | controverso | case | tips'
cta_final: 'salve | comente | link na bio | siga'
```

---

## Process

### 1. Estrutura (@content-producer)

- Define hook (slide 1)
- Estrutura conteudo (slides 2-8)
- CTA final (slide 9-10)
- Caption + hashtags

### 2. Design (@visual-designer)

- Layout visual por slide
- Cores e tipografia do design system
- Specs: 1080x1350px (4:5)

### 3. Quality Gate

- Checklists: content-quality + brand-consistency + mobile-first

---

## Output

```
Por slide:
- Texto principal
- Texto secundario
- Layout (imagem/icone posicao)

Extras:
- Caption completa
- 20-25 hashtags
- Sugestao de horario
```
