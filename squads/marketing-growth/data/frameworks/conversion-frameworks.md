# Conversion Frameworks

## 1. Cialdini - 6 Principios de Persuasao

- Reciprocidade: de valor primeiro
- Compromisso: pequenos "sins" levam a grandes
- Prova Social: depoimentos, numeros, logos
- Autoridade: credenciais, dados, parceiros
- Simpatia: storytelling, humor, vulnerabilidade
- Escassez: APENAS real, nunca fabricada

## 2. Fogg Behavior Model (B = MAP)

Behavior = Motivation + Ability + Prompt

- Motivation alta + Acao facil + CTA claro = Conversao

## 3. MECLABS Heuristic

C = 4M + 3V + 2(I-F) - 2A

- M (4x): Motivacao do usuario
- V (3x): Clareza da proposta de valor
- I-F (2x): Incentivo menos Friccao
- A (2x neg): Ansiedade

## 4. PIE Framework (Priorizacao CRO)

| Fator      | Pergunta                    |
| ---------- | --------------------------- |
| Potential  | Quanto pode melhorar?       |
| Importance | Quao importante e a pagina? |
| Ease       | Quao facil e testar?        |

Score = (P + I + E) / 3

## 5. Quick Check LP (10 pontos)

1. Headline clara em 5s?
2. CTA above the fold?
3. Beneficio > Feature?
4. Social proof presente?
5. Garantia explicita?
6. Sem navegacao (foco total)?
7. Mobile-first?
8. Form minimo?
9. Speed < 3s?
10. Tracking instalado?
