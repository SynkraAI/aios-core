# Storytelling Frameworks

## 1. Before-After-Bridge (BAB)

BEFORE: situacao atual (dor)
AFTER: como seria sem a dor
BRIDGE: seu produto como ponte
**Melhor para:** Posts curtos, emails, ads

## 2. Hero's Journey

1. Mundo Comum > 2. Chamado > 3. Recusa > 4. Mentor (voce)
2. Cruzar Limiar > 6. Provacoes > 7. Caverna > 8. Recompensa > 9. Retorno
   **Melhor para:** Sales pages, VSLs, webinars

## 3. StoryBrand (Donald Miller)

1. CHARACTER: cliente e o heroi
2. HAS A PROBLEM: externo, interno, filosofico
3. MEETS A GUIDE: voce (empatia + autoridade)
4. WHO GIVES A PLAN: 3 steps simples
5. CALLS TO ACTION: direto e transitional
6. AVOIDS FAILURE: consequencias
7. ENDS IN SUCCESS: transformacao
   **Melhor para:** Landing pages, homepage

## 4. PASTOR (Ray Edwards)

P-Problem, A-Amplify, S-Story/Solution, T-Transformation, O-Offer, R-Response
**Melhor para:** Emails de venda

## 5. AIDA

A-Attention, I-Interest, D-Desire, A-Action
**Melhor para:** Ads, posts curtos

## Quando Usar Qual

| Contexto       | Framework                   |
| -------------- | --------------------------- |
| Post Instagram | BAB ou AIDA                 |
| LinkedIn       | BAB ou StoryBrand           |
| Landing page   | StoryBrand + Hero's Journey |
| Email          | PASTOR ou BAB               |
| VSL            | Hero's Journey              |
| Ad copy        | AIDA                        |
