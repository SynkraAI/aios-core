# Mobile-First Checklist

## Layout

- [ ] Design responsivo (funciona em 375px)?
- [ ] Sem scroll horizontal?
- [ ] Touch targets 44x44px minimo?

## Tipografia

- [ ] Texto legivel sem zoom?
- [ ] Tamanho minimo 16px?

## Imagens

- [ ] Imagens otimizadas (max 200KB)?
- [ ] Aspect ratio correto por plataforma?
- [ ] Texto sobre imagem legivel no mobile?

## Performance

- [ ] Pagina carrega em menos de 3s?
- [ ] CSS inline para critical path?

## Formularios

- [ ] Campos com tamanho adequado?
- [ ] Keyboard adequado (email, tel, text)?
- [ ] Botao CTA visivel sem scroll?
- [ ] Max 3-5 campos?

## Landing Pages

- [ ] Hero section impactante no mobile?
- [ ] CTA above the fold?
- [ ] FAQ com accordion?

---

Testar em: iPhone SE (375px), iPhone 14 (390px), Galaxy (360px)
