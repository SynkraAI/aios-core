# SEO Basics Checklist

## On-Page

- [ ] Title tag unico (50-60 chars)?
- [ ] Meta description atraente (150-160 chars)?
- [ ] H1 unico na pagina?
- [ ] H2-H6 hierarquia correta?
- [ ] URL amigavel (curta, kebab-case)?
- [ ] Keyword no H1, title e meta description?

## Conteudo

- [ ] Conteudo original?
- [ ] Extensao adequada (min 300 words para LP)?
- [ ] Keyword density natural (1-2%)?

## Tecnico

- [ ] Imagens com alt text?
- [ ] Imagens comprimidas?
- [ ] Pagina responsiva?
- [ ] Carregamento rapido (< 3s)?

## Social

- [ ] Open Graph tags (og:title, og:description, og:image)?
- [ ] Imagem de compartilhamento (1200x630px)?
