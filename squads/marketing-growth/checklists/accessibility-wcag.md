# Accessibility WCAG Checklist

## Contraste

- [ ] Texto normal: contraste 4.5:1 minimo?
- [ ] Texto grande (24px+): contraste 3:1 minimo?
- [ ] Elementos interativos com contraste adequado?

## Tipografia

- [ ] Fonte legivel (nao decorativa para corpo)?
- [ ] Tamanho minimo 16px para corpo?
- [ ] Line-height minimo 1.5?

## Imagens e Midia

- [ ] Alt text em todas as imagens?
- [ ] Legendas em videos?
- [ ] Nao depende so de cor para comunicar info?

## Navegacao (Landing Pages)

- [ ] Headings hierarquicos (H1 > H2 > H3)?
- [ ] Links com texto descritivo?
- [ ] Formularios com labels?
- [ ] Tab navigation funcional?
- [ ] Focus states visiveis?

## Conteudo

- [ ] Linguagem clara?
- [ ] Frases curtas e diretas?

---

Standard: WCAG 2.1 Level AA
